What is Computer Science

If you want to study with us, it is important that you understand the difference between Computer Science and Information Technology

Computer Science

Students of Computer Science typically want to create and design software or systems to allow others to achieve a goal. The focus of their studies will include learning how a computer works and how to program or configure it to perform new tasks by creating custom solutions i.e. a new program/app or building a computer network or system. Computer Science students are commonly know as computer practitioners.

Information Technology

Students of Information Technology typically study subjects that involve purely the use of computer application. The focus of their studys is to learn to use software, for example an IT student will spend time learning to use a word-processor with the aim of being able to word-process well laid out documents without really wanting to know anything more that what features the software has. IT students are commonly known as computer users.

Computing Learning Labs

For our students it is important that we provide a range of learning areas to support the curriculum we deliver within the college, the department has a wide range of computing equipment distributed in a number of labs.

Whether you want to be a programmer, techician, network engineer, database admin, web developer, security specialist or games programmer our labs have all the facilities neccessary for you to study these subjects.

We are constantly looking to improve and extend our facilities to support new learning opportunities, it is unlikely that you will see the same range of facilites in school and the smaller colleges in the area.

Computer Servicing Labs

Our servicing labs at Lincoln and Newark provide all the facilites for computer servicing including dedicated i7 and i5 PC's for performing tasks such as component replacement, operating system and software installation. Supporting this is a dedicated network that allows us to change every computers configuration in 20 minutes allowing you to work with a range of operating systems including all versions of windows and different versions of Linux.

The tasks performed within the labs and their configurations have been designed to emulate the working environment of the KNOWHOW® national repair centre.

Security and Networking Lab

The networking lab provides students with equipment to understand the principles of computer networking in modern business this includes dedicated servers, PC's, routers and switches all connected to the sections custom private network. This allows students to set-up and run their own network configurations and configure network devices within the network testing security and performance using specialist software tools such as wireshark.

The design of the labs was influenced by staff within the department who have practical industrial networking experience and CISCO® qualifications.

More Computer Networking Information

Web Development Lab

The web development lab provides students with their own customisable web servers through virtual technologies, enabling the development of live websites from day one of their studies, the system allows the student to experiment with, HTML 5, CSS, JavaScript, PHP, Python, NodeJS and their own online MySQL databases to produce industry standard data-driven websites.

Once students have set-up their servers they can access and edit from any computer that provides a web browser.

More Computer Networking Information

Games Development Lab

The games development lab features its own internal development network with PC's capable of running both windows and linux operating systems and a full suite of standard programming systems as part of the main college network allowing for flexible working within the room. PC games controllers form part of the extra hardware within the lab allowing students to experiment with different control systems.

More Computer Networking Information

Programming Lab

The programming lab provides students with a range of programming tools that allow development of software using Microsoft C#, Java, Python, C++, Haskell, Prolog. As part of joining the computer science department you will be entitled to download and use the same tools at home as well as at college.

The lab is intended to be a quiet workspace with modern computer systems that use solid state drives to increase the speed of compilation of code to ensure you get from code to running program as quickly as possible.
